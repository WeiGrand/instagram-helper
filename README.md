# Instagram Helper

My Instagram helper

```bash
$ git clone git@github.com:WeiGrand/instagram-helper.git

$ cd instagram-helper
```

## Already done

- Mock login

  ```bash
  $ npm run login
  ```
  
  > You can fill your username/password in `account.json` to avoid typing each time.

- Scrape all the medias(photo/video) form user's homepage

  ```bash
  $ npm run scrape-user
  ```
